IoT · Infrastructure · Venture Pitch In Iteration · Phase 1

DigestiNet — IoT
Biodigester Network

Centralized smart-waste infrastructure · Co-founder & Systems Design

A centralized IoT network that turns organic market waste into clean biogas and bio-fertilizer, with real-time telemetry across every digester node. Designed as a proof-of-concept for Mukono Market, with a scaling model for high-density commercial hubs across East Africa.

  • Sensor network + real-time telemetry across digester nodes
  • Centralized dashboards for technicians and the community
  • Problem-tree analysis, revenue model, and rollout roadmap

Learning & iteration

First-round stakeholders found the architecture hard to read, so the current cycle is focused on simplifying the telemetry into plain, accessible insight for non-technical market vendors. Phase 2 pitch lands at the end of this recess semester.

Web App · React · Supabase · Personal Project In Development

Executive Tracker

Productivity & Behavioral Management Platform

A personal project born out of a real problem: most productivity tools let you set goals but fail to show whether you're actually hitting them. The Executive Tracker bridges that gap — converting daily habits and routines into structured data you can actually interrogate, so you replace "I think I was productive" with evidence.

  • Quantifies daily habits into measurable performance metrics
  • Detects "drift" — deviations from your established routines over time
  • Postgres data store via Supabase, with row-level security and JWT auth
  • React + Vite + Tailwind front end, querying Supabase directly from the client
Building in progress

The interface is still being built — a live demo and screenshots land here once it's ready to show properly.
